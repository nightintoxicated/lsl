Dependencies: uses perl!


to install:

<b>Option 1:</b>

make sure you have make installed

git clone the repo , cd into the cloned directory, type <b>make install</b>


<b>Option 2:</b>

download the file, chmod +x lsl, then move it to /bin or another $PATH
